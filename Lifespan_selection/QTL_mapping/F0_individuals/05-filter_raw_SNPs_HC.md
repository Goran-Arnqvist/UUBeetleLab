# 2023-12-13

The main filtering steps I will follow are:

1. **GATK hard-filtering (hf)**: The goal is to obtain a data set containing only variant sites that we trust with high confidence. Then, we will remove variants that have skewed GATK quality annotation values across all samples
2. **DP and GQ**: Filter on minimum and maximum read depth (DP) and genotype quality (GQ)
3. **Monomorphic, missingness, and gap around indels**: Filter out monomorphic SNPs, samples with a high amount of missing data, variants with a high amount of missing genotypes, and filter on minor allele frequency

#!/bin/bash
#SBATCH -A snic2020-5-94
#SBATCH -M snowy
#SBATCH -p core -n 4
#SBATCH -t 20:00:00
#SBATCH -J apply_DPfilter
#SBATCH -e apply_DPfilter_%J_%A_%a.err
#SBATCH -o apply_DPfilter_%J_%A_%a.out
#SBATCH --mail-type=all
#SBATCH --mail-user=angela.fuentespardo@gmail.com
#SBATCH --mail-type=ALL
#SBATCH -a 1-3

# Load required programs in the cluster computer.
module load bioinfo-tools
module load vcftools/0.1.15
module load bcftools/1.10
module load samtools/1.10
module load zlib/1.2.11

# Set environment variables.
# Files.
WORK_DIR='/proj/snic2020-6-128/private/a_obtectus_QTLmap/parentals/data/04-vcf-files'
INPUT_FILE='/proj/snic2020-6-128/private/a_obtectus_QTLmap/parentals/data/04-vcf-files/OTT_8_F0.HC.SNPs.hf.GQ20.mono.miss20.mac3.vcf.gz'
SNP_LIST='./list_SNPs_passed_filters.txt'
OUTPUT_PREFFIX='./list_output_filenames.txt'

# Go to the SNIC_TMP (working) directory.
cd $WORK_DIR

# Set path to files specific to each job array.
echo This is array job: $SLURM_ARRAY_TASK_ID
SNP_target=$(sed -n "$SLURM_ARRAY_TASK_ID"p $SNP_LIST)
OUTFILE_target=$(sed -n "$SLURM_ARRAY_TASK_ID"p $OUTPUT_PREFFIX)
echo -e This is the file listing target SNPs: ${SNP_target}
echo -e This is the output filename: ${OUTFILE_target}

# 1) Apply GATK hard quality filters and FORMAT filters (missing rate, MAC). (Done previously)

# 2) Get VCF with only the SNPs that passed the DP filter.
# Reformat the file listing SNP loci to have one column for chromosome and another one for position, separated by a tab.
awk -F'-' -v OFS='\t' '{print $1,$2}' $SNP_target > $OUTFILE_target.filtered_SNPs.list

# Generate a VCF file with only the filtered SNPs.
vcftools --gzvcf $INPUT_FILE --positions $OUTFILE_target.filtered_SNPs.list --recode --recode-INFO-all --stdout | bgzip > $OUTFILE_target.vcf.gz
# Rename the resulting file (remove the "recode" part)
#mv $OUTFILE_target.recode.vcf $OUTFILE_target.vcf

# Generate an index for the final VCF file.
tabix -p vcf $OUTFILE_target.vcf.gz

# Number of SNPs in the filtered file.
echo -e "Number of SNPs in $OUTFILE_target.vcf.gz is":
bcftools view -H $OUTFILE_target.vcf.gz | wc -l
